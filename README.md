# 2048 на C++

Консольная реализация игры 2048, разработанная на языке C++ с применением методологий TDD (Test-Driven Development) и BDD (Behavior-Driven Development).

##  Описание проекта

Игра представляет собой реализацию классической логической головоломки 2048 с игровым полем размером 4×4.

Игрок управляет перемещением чисел с помощью стрелок клавиатуры.  
При каждом ходе:

- На поле появляется новая плитка со значением 2
- Одинаковые числа при столкновении объединяются
- Значения суммируются
- Начисляются очки

Для победы необходимо собрать число 2048.  

Если поле заполнено и отсутствуют возможные ходы — игрок проиграл.
